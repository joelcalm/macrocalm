# Current Workout Plan

## Goal context

Primary goal: planche.

Secondary goals:
- Improve front lever.
- Front lever pull-up.
- 90-degree push-up.
- Improve running/cardio.

The plan is not meant to be followed perfectly every day. The app should track what was planned versus what was actually completed, including substitutions, skipped sets, easier/harder variations, pain/fatigue notes, and extra work.

---

## MON — PUSH A: Hard Planche + 90° Negatives

**Cap: 70–75 min**

### Warm-up — 10–12 min

Wrists, shoulders, scap push-ups, hollow body, light planche leans.

### Main work

1. **Advanced tuck planche touch holds** — 2×3–5s  
   Rest 3 min.

2. **Tuck planche holds** — 4×6–10s  
   Rest 2:30–3 min.  
   Reduced from 5 sets because there is now more weekly planche frequency.

3. **Planche leans** — 3×15–25s  
   Rest 90–120s.

4. **Pseudo planche push-ups** — 3×4–8  
   Rest 2:30–3 min.

5. **90° push-up negatives** — 3×2–3 @ 4–6s down  
   Rest 3 min.  
   Keep these very clean. Do not turn them into ugly shoulder survival reps.

6. **Compression / hollow hold** — 2–3 sets  
   Rest 60–90s.

### Optional FL micro — 5–6 min only

Only if elbows/shoulders feel good.

- Easy tuck FL hold — 2×8–10s
- Scap pull-up or scap depression hold — 2×6–10

This is not a workout. It is just a small technical exposure.
